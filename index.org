#+title: ⚙ cogs ⚙
#+subtitle: Configuration Object Generation System
#+setupfile: setup.org

⚙ ~cogs~ ⚙ is a system for generating code for the processing of
configuration objects based on defining abstract schema which may be
transformed into specific generating schema (eg, for Avro, Protobuf,
etc) or validating schema (eg JSON Schema).
